## Installation
- git clone git@github.com:jasonkaplan79/laravel_vidstack.git 
- cd laravel_vidstack
- composer install 
- php artisan migrate --seed 
- npm install 
- npm run build 
- php artisan serve 
- Go to the URL like http://localhost:8000, http://127.0.0.1:8000 etc.
- Go to /test1
- Go to /test2
